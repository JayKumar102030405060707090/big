# Telegram Big/Small Bot

यह बॉट चैनल और ग्रुप में काम करता है और केवल एडमिन द्वारा भेजी गई कमांड पर प्रतिक्रिया देता है।

## सपोर्टेड कमांड्स:

- `/big` → Big
- `/small` → Small
- `/red` → Red
- `/green` → Green
